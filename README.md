# CSV Data Extraction
## Script para Extração de Dados de arquivos CSV

## Instalação

Requisito: [Go Lang](https://golang.org/) v1.16.3+ para executar.

Instalação das dependências e iniciar a executação do script.

```sh
cd extract-data-csv
go get -u ./...
```

## Execução do script

```sh
go run main.go
```
